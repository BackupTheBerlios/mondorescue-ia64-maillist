There is no pre-builder rpm package for RHEL5 :-(

So you need to use ftp://ftp.mondorescue.org/src/afio-2.4.7-1.src.rpm
and ftp://ftp.mondorescue.org/src/buffer-1.19-1.src.rpm in order to
build the on ia64 (using receipt at
http://trac.mondorescue.org/wiki/DistributionPackaging#RPMrebuildprocess)
